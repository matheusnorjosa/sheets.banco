# Code of Conduct

This project follows the principles of the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/). The full text is canonical and supersedes anything here.

## TL;DR

- Be respectful, inclusive, and assume good intent.
- Critique ideas, not people.
- Keep discussions focused on technical and project topics.
- Sensitive topics outside the project's scope belong elsewhere.

## Reporting issues

If someone's behavior in the project is making you uncomfortable, please email **matheusnorjosa@gmail.com**. Reports are kept confidential and acted on in good faith.

Maintainers may remove comments, commits, code, issues, or pull requests that violate this standard, and may temporarily or permanently restrict contributors who repeatedly do so.

## Scope

This applies to all project spaces — GitHub repository (issues, PRs, discussions), and any related communication channels we set up.
